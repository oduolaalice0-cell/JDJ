# JDJ AI Studio Preview App

This repository contains an interactive static preview of a **HeyGen-style** AI video web app concept for JDJ AI Studio.

## Features in this preview

- Sidebar app layout (dashboard-style)
- Login modal flow
- Prompt-to-video input section
- Template picker with active selection
- Avatar, Projects, and Settings preview pages
- Brand logo block + support/contact details

## Run locally

```bash
python3 -m http.server 4173
```

Open: `http://localhost:4173`
